# Dataset Notes

This repository includes local parquet copies of the dataset splits used by the experiments:

- CLINC-OOS: `dataset/clinc_oos/`
- BANKING77: `dataset/banking77/`

The code license for this repository does not change the license of the datasets. Please follow the original dataset licenses and citation requirements when reusing the data.

## CLINC-OOS

Source: https://huggingface.co/datasets/clinc/clinc_oos

The Hugging Face dataset card currently lists the license as `cc-by-3.0`.

## BANKING77

Source: https://huggingface.co/datasets/PolyAI/banking77

The dataset metadata lists the original data license as Creative Commons Attribution 4.0 International. The Hugging Face dataset loading script has its own Apache-2.0 notice, but that should not be confused with the data license.
